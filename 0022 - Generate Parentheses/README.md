# 22. Generate Parentheses

```yaml
Link: https://leetcode.com/problems/remove-nth-node-from-end-of-list/
Difficulty: Easy
Topics:
  - Backtracking
  - Dynamic Programming
  - String
```

Given n pairs of parentheses, write a function to generate all combinations of well-formed parentheses.



<br>

# Example 1:

```yaml
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```

<br>

# Example 2:

```yaml
Input: n = 1
Output: ["()"]
```

<br>

# Constraints:

```yaml
- 1 <= n <= 8
```
